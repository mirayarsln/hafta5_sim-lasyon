Çalışmanın İçeriği
Excel dosyasında yer alan veri setinde:
- X (Çalışan Sayısı)
- Y (Üretim Sayısı)  
değişkenleri kullanılmıştır. Korelasyon katsayısının hesaplanabilmesi için aşağıdaki ara değerler oluşturulmuştur:
- ( X^2 ) (Çalışan sayısının karesi)
- ( Y^2 ) (Üretim sayısının karesi)
- ( X*Y ) (çarpım değerleri)

Değerler kullanılarak Pearson Korelasyon Katsayısı (r) hesaplanmıştır.

- Korelasyon Katsayısı (r): `0,869`

Bu sonuç, çalışan sayısı ile üretim miktarı arasında güçlü ve pozitif bir ilişki olduğunu göstermektedir.
Yani çalışan sayısı arttıkça üretim miktarı da artma eğilimi göstermektedir.

Bu çalışmanın amacı:
- İki değişken arasındaki ilişkiyi sayısal olarak ölçmek  
- Korelasyon hesabının mantığını Excel üzerinden uygulamalı olarak göstermek  
- Simülasyon ve istatistiksel analiz konularını birlikte değerlendirmektir
